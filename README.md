# Notes Management System

A web-based Notes Management System developed using PHP, MySQL, HTML, and CSS.  
This project allows users to create, edit, delete, and manage notes securely.

## Features
- User authentication (Login & Register)
- Create, edit, and delete notes
- MySQL database integration
- Clean and responsive UI
- Secure CRUD operations

## Technologies Used
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Version Control: GitHub

## Project Status
- Completed
- Not hosted live (source code available)
